Projects

A collection of projects created to highlight my skillset

Incremental Fighter

A collaborative, MERN stack project that tasks players with defeating enemies by clicking - growing stronger all the while!

Incremental Fighter features React as templating language, MongoDB for saving and loading a users progress after authentication through Google and Firebase, Node.js and Express on the server side, Bootstrap and CSS for styling, React-Audio-Player for the music, and is fully deployed on Heroku.

See it in Action! Git Hub Repo

Munchies

A collaborative project that allows users to tap into Google Places API and the Eat Street API to grab two things that go incredibly well together: Weed and Food!

Created with HTML5, CSS3, Symantic UI, Firebase, Javascript, jQuery, and AJAX

See it in Action! Git Hub Repo

Friend Finder

An Heroku hosted application. Users fill out a basic questionaire and are matched with the closest response from other users!

Created with Node.js, Express, Bootstrap, JSON, and Javascript.

See it in Action! Git Hub Repo

Train Scheduler

An exercise to develop an example train scheduler. Users can input their departure time, the name of the train, and will recieve outputs of the next train arrival time and how frequently the train appears.

Uses Moment.js, HTML5, Bootstrap, Javascript, jQuery, and Firebase.
